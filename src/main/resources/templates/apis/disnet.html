DISNET REST API

Getting Started

Base URL

disnet.ctb.upm.es/api/disnet

Schemes

HTTP | HTTPS

Requirements

El requisito que se necesita para hacer uso de esta API es tener una cuenta DISNET. Cuando se genera una cuenta se proporciona un token de acceso (identificador único). El token siempre debe incluirse en cada petición que se realice a un servicio, si se encuentra activo el servicio responderá a la petición realizada, sino no se visualizarán los resultados.

El token se obtiene registrándose en la aplicación Web de DISNET, que se encuentra aquí.

Quick Start

Resumen del uso de la API.

  • 1. Seleccionar herramienta. Para visualizar los resultados de la peticiones existen dos formar de hacerlo: 1) con un visualizador Web, utilice el de su preferencia (Opera, Chrome, Firefox, Safari, IE, entre otros) y 2) alguna herramienta amigable al usuario para trabajar con APIs, se recomienda el uso de la herramienta Postman (se encuentra disponible para usuarios: Windows, Mac OS X, Linux y Chrome que se descarga en el siguiente enlace: https://www.getpostman.com).
  • 2. Seleccionar servicio de interés dentro de la lista de servicios oficiales de DISNET.
  • 3. Formar petición definiendo los parámetros de búsqueda solicitados según cada servicio.
  • 4. Realizar petición.
  • 5. Ver resultados.
